github repository download.. git clone link to your repository
git init - initialize local git repository
git status - status of stage of the files that is located in git repository
git add {file name} - fisrt stage of pushing a file into the git repository
git commit -m "comment" - last stage 
git remote add https://github.com/drsc1ence/PP2-Damir-.git - this can be used only once, in order to connect local repository with global
git push -u origin main - push changes into global repository
git remote set-url origin NEW-LINK - local repository link to new global repository name
git log - see history of commits
git restore file name - restore unstaged file
git restore --staged file name - restore staged file

git add . - adds all files in folders 
mkdir {folder name} - creates a folder
pwd - checks my position 
cd {folder name} - enters in the folder
cd.. - goes back
